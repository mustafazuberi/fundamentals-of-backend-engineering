# Fundamentals of Backend Engineering

This repo is my workspace for:
- **Notes** (concepts, pitfalls, diagrams)
- **Mini-projects** (small runnable demos)
- **Exercises** (practice + solutions)

## Structure
- `notes/` — course notes, one topic per file
- `projects/` — runnable demos grouped by topic
- `exercises/` — small prompts + solutions

## How I’ll work through a topic
1. Create a note in `notes/` (template below).
2. Add a runnable demo in `projects/<topic>/`.
3. Add 3–10 exercises in `exercises/<topic>/`.

## Note template
Copy this into a new file in `notes/`:

```
# Topic

## What problem does it solve?

## How it works (high level)

## Request/response flow (step-by-step)

## Tradeoffs
- Pros:
- Cons:

## Failure modes / gotchas

## When to use / avoid

## Related concepts

## References (optional)
```
